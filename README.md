📐 Geometry Genius

🔗 Live Site: https://geometry-genius.vercel.app/

A simple and interactive geometry calculator web application that allows users to calculate areas of different shapes dynamically. Built with a focus on clean UI, usability, and real-time calculation.

🚀 Features
📏 Calculate area of different geometric shapes
⚡ Real-time calculation and result display
🎯 Input validation for accurate results
🧮 Calculation history tracking
📱 Fully responsive design
🎨 Interactive UI with dynamic elements


🛠️ Technologies Used
HTML5
CSS3 / Tailwind CSS
JavaScript (Vanilla JS)

├── scripts/              # JavaScript files
├── index.html            # Main HTML file
├── tailwind.config.js    # Tailwind configuration

⚙️ Setup & Usage

Clone the repository:

git clone https://github.com/istewakhassantewak/geometry-genius.git

Navigate to the project folder:

cd geometry-genius
Open index.html in your browser.
